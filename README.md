# QABot

Сборка: `docker build -t qabot .`  
Запуск: `docker run -d --name QABot -p 80:80 qabot`  
Остановка: `docker stop QABot`  
Удаление: `docker rm QABot`  
Тест:  

- GET-запрос `http://127.0.0.1/`:

  ```bash
  curl -X 'GET' \
    'http://127.0.0.1/' \
    -H 'accept: application/json'
  ```

  Ответ:

  ```json
  {
    "response": "Welcome!"
  }
  ```
